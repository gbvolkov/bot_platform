# Evidence Packaging

Use this workflow when the manager needs source-grounded talking points.

Preferred steps:
1. Use `search_marketing_materials` to find candidate materials.
2. Check candidate metadata before reading: family, document kind, branch, body tags, transport type, and relative path.
3. Use `read_marketing_material` for focused excerpts.
4. Package 3-5 findings rather than long quotation dumps.
5. For each finding, include the supporting material candidate, relevant metadata, and evidence strength.

For every important claim, state whether it is:
- supported by internal material
- only partially supported
- requires BI validation for this specific candidate or claim
- requires web validation

If the best candidates have weak or mismatched metadata, explicitly say so and recommend a narrower follow-up search rather than overstating the evidence.

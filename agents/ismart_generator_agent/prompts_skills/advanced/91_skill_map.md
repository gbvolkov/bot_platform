# Skill Map генератора УМК

Основной JSON: `../для_теста_существенные_данные.json`

Референсы берутся из `lesson.materials_md.*` внутри JSON. Все skill-файлы ниже работают от JSON и Markdown-референсов.

| Skill file | Skill id | Когда применять | Основной уровень данных | Выход |
|---|---|---|---|---|
| `01_Общее_prompt_skill.md` | `umk_common_generation_rules` | Всегда перед генерацией любого материала УМК по данным JSON. | course/module/lesson | глобальные правила генерации УМК и маршрутизация артефактов |
| `02_Теория_prompt_skill.md` | `theory_material_generation` | Используй для занятий, где `lesson.content_flags.theory = true`; для ТП генерируй теорию отдельным файлом. | lesson | ученический материал теории и МР-теория |
| `03_Практика_prompt_skill.md` | `practice_material_generation` | Используй для занятий, где `lesson.content_flags.practice = true`; для ТП генерируй практику отдельным файлом. | lesson | ученическая практика, тест-кейсы, ключи в МР/QA |
| `04_Самостоятельная_prompt_skill.md` | `self_study_generation` | Используй для занятий, где `lesson.content_flags.self_work = true`. | lesson | самостоятельная работа и самоконтроль |
| `05_Промежуточная_prompt_skill.md` | `intermediate_assessment_generation` | Используй на уровне модуля, если хотя бы одно занятие имеет `lesson.content_flags.attestation = true`. | module | промежуточная аттестация v34: 4 × [5 тестовых + 5 open-code + 5 code tasks] |
| `06_Итоговая_prompt_skill.md` | `final_project_assessment_generation` | Используй на уровне курса, когда требуется итоговая аттестация по всему JSON-курсу. | course | итоговая аттестация в формате проекта |
| `07_Методические_указания_prompt_skill.md` | `teacher_method_guidance_generation` | Используй вместе с теорией, практикой и аттестацией; не используй для самостоятельной работы. | lesson | методические указания учителю |
| `08_Форматирование_заданий_курса_prompt.md` | `course_task_formatting_prompt` | Используй как обязательный слой финального оформления для всех материалов, заданий, вопросов, тест-кейсов, ключей и методических пояснений. | course/module/lesson | правила форматирования материалов курса |

## Recommended routing
- Любая генерация начинается с `01_Общее_prompt_skill.md` и всегда подключает `08_Форматирование_заданий_курса_prompt.md`.
- Если `lesson.content_flags.theory = true`, добавь `02_Теория_prompt_skill.md`.
- Если `lesson.content_flags.practice = true`, добавь `03_Практика_prompt_skill.md`.
- Если `lesson.content_flags.self_work = true`, добавь `04_Самостоятельная_prompt_skill.md`.
- Если `lesson.content_flags.attestation = true`, работай на уровне модуля и добавь `05_Промежуточная_prompt_skill.md`.
- Для итоговой проектной аттестации по курсу используй `06_Итоговая_prompt_skill.md`.
- Для МР к теории/практике/аттестации добавь `07_Методические_указания_prompt_skill.md`.
- Для занятия типа ТП теория и практика генерируются отдельными ученическими файлами; МР тоже разделяется по теории и практике.
